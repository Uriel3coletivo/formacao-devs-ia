Formação Devs IA - Site Institucional
Site desenvolvido em Next.js 14 com Tailwind CSS e Framer Motion.

📋 Pré-requisitos
Node.js 18+ instalado
npm ou yarn
🔧 Instalação
Clone o repositório:
git clone https://github.com/Uriel3coletivo/formacao-devs-ia.git
cd formacao-devs-ia

Instale as dependências:
npm install

ou

yarn install

▶️ Executar localmente
npm run dev

ou

yarn dev

Abra http://localhost:3000 no navegador.

🌐 Deploy na Vercel (RECOMENDADO)
Opção 1: Via GitHub (automático)
Acesse vercel.com
Faça login com GitHub
Clique em "Add New..." > "Project"
Selecione o repositório "formacao-devs-ia"
Clique em "Deploy" (configuração automática)
Opção 2: Via CLI
npm install -g vercel
vercel login
vercel --prod
